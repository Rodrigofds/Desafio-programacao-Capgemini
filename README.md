###### Questão 01
# Desafio Academia Capgemini 2022

##### Como executar as Questões:

	Clone o projeto em sua IDE de preferência;
	Clique com o botão direito em cima da questão escolhida -> Run As -> Java Application.

##### Para executar os Testes Unitários:

	Clique com o botão direito em cima do teste escolhido -> Run As -> JUnit Test.

**Obs.:** Necessário o JUnit 5 instalado.

###### Tecnologias utilizadas no desafio:

	Java 11
	IDE Eclipse
	JUnit5
	Javadoc
